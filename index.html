About Me

Machine Learning Researcher | ADAS Software Engineer | Ph.D. in Mechanical Engineering

Passionate about advancing autonomous systems

Work Experience

At Canoo, my role as an ADAS Software Engineer involves delivering advanced control systems and sensor fusion techniques to enhance vehicle safety. In this role, I have strengthened my programming skills in C++ and Python and gained hands-on experience in software development within the automotive industry: Design, test, optimize and deploy a real-time Joint probabilistic data association-based multi-sensor multi-object tracking and data fusion algorithm for ADAS system Develop simulation/replay-based pipelines for tracker evaluation and tuning, accelerating the software development lifecycle and ensuring optimal performance Collaborate with cross-functional teams to integrate sensor fusion solutions, ensuring reliability through SIL, HIL, and real-world testing

Research

  • Created a Machine Learning model for multi-modal pedestrian trajectory prediction using visual-sensing data
  • Led a team for developing a real-time perception-feedback UAV system for human-robot interaction in the wild
  • Constructed a visual-sensing-based fast and accurate action detection framework for automatically recognizing distinct actions from untrimmed action sequence
  • Collaborated with an interdisciplinary team from CMU and GaTech for the stochastic distributed optimal dual control project
